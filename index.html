Introdução ao HTML

Linguagem de Marcação de Hipertexto - Estrutura de Documentos, Formatação e Links

O que é HTML?

HTML (HyperText Markup Language) é a linguagem fundamental para criação de páginas web. Ela define a estrutura e o conteúdo de um site, utilizando elementos ou tags para marcar diferentes partes do conteúdo.

HTML não é uma linguagem de programação, e sim uma linguagem de marcação que define a estrutura do seu conteúdo.

Estrutura Básica de um Documento HTML

Todo documento HTML segue uma estrutura básica composta pelos seguintes elementos:

<!DOCTYPE html>
<html>
<head>
  <title>Título da Página</title>
</head>
<body>
  <!-- Conteúdo da página aqui -->
</body>
</html>

Principais Tags de Formatação

Cabeçalhos

Os cabeçalhos vão de <h1> (maior importância) a <h6> (menor importância):

Isso é um H1

Isso é um H2

Isso é um H3

Isso é um H4

Isso é um H5
Isso é um H6

Parágrafos e quebras de linha

<p>Isso é um parágrafo.</p>
<p>Isso é outro parágrafo com <br><br>quebra de linha.</p>

Formatação de texto

Texto em negrito com <strong> ou <b>

Texto em itálico com <em> ou <i>

Texto sublinhado com <u>

Texto tachado com <s>

Links em HTML

Links são criados com a tag <a> (âncora) e o atributo href que especifica o destino do link.

<a href="https://www.exemplo.com">Isso é um link</a>
<a href="pagina2.html" target="_blank">Link que abre em nova aba</a>

Exemplos de links:

Link interno (âncora) Link externo (Google) Link de e-mail

O atributo target="_blank" faz com que o link seja aberto em uma nova aba/janela.

Estrutura de Documentos HTML

Além da estrutura básica, documentos HTML modernos utilizam tags semânticas para melhor organização:

<header><!-- Cabeçalho --></header>
<nav><!-- Menu de navegação --></nav>
<main>
  <section><!-- Seção de conteúdo --></section>
  <article><!-- Artigo --></article>
</main>
<footer><!-- Rodapé --></footer>

Exemplo Completo

Aqui está um exemplo completo de uma página HTML simples:

<!DOCTYPE html>
<html lang="pt-BR">
<head>
  <meta charset="UTF-8">
  <title>Minha Primeira Página</title>
</head>
<body>
  <header>
    <h1>Bem-vindo ao Meu Site</h1>
  </header>

  <nav>
    <a href="#home">Home</a> |
    <a href="#sobre">Sobre</a> |
    <a href="#contato">Contato</a>
  </nav>

  <main>
    <section id="home">
      <h2>Introdução</h2>
      <p>Este é o conteúdo da seção inicial.</p>
    </section>
  </main>

  <footer>
    <p>© 2023 Meu Site. Todos os direitos reservados.</p>
  </footer>
</body>
</html>
